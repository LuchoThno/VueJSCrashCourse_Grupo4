# Trekking Adventures

Vue.js project for Trekking Adventures website.

## Project Setup

```bash
npm install
```

### Compile and Hot-Reload for Development

```bash
npm run dev
```

### Compile and Minify for Production

```bash
npm run build
```

## Project Structure

- `src/` - Source files
  - `components/` - Vue components
  - `views/` - Page views
  - `router/` - Vue Router configuration
  - `assets/` - Static assets (CSS, images)

## Tech Stack

- Vue 3
- Vue Router
- Vite
